# 快乐不起来的Linux命令

## 文件系统中跳转

> pwd(print work directory) 打印当前工作目录名
>
> cd (change dircetory) 
>
> ls (list) 列出目录内容

* 绝对路径:开始于根目录,紧跟者目录树的一个个分支,一直到达期望的目录或文件
* 相对路径:开始于工作目录,一对特殊符号(`.`和`..`)来表示相对位置.

### cd快捷

| 快捷            | ?                        |
| --------------- | ------------------------ |
| cd              | 回到当前用户的home目录下 |
| cd -            | 回到上一个去的目录       |
| cd ~<user_name> | 回到别人家的home目录     |

> # Linux文件的一些规则
>
> 1. `.`开头的文件是隐藏文件,`ls`命令看不到,要用`ls -a`
> 2. 文件名和命令名大小写敏感.
> 3. Linux没有"文件扩展名"的概念.
> 4. Linux文件名标点符号支持`.`,`-`,`_`,最好还是不要用空格,用下划线.
